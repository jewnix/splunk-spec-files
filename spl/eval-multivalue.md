| Function | Syntax | Description | Usable With | Docs |
|----------|--------|-------------|-------------|------|
| `commands` | `commands(<value>)` | This function takes a search string, or field that contains a search string, and returns a multivalued field containing a list of the commands used in <value>. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title2) |
| `mvappend` | `mvappend(<values>)` | This function takes one or more values and returns a single multivalue result that contains all of the values. | `mvcount` | `mvcount(<mv>)` | This function takes a field and returns a count of the values in that field for each result. If the field is a multivalue field, this function returns the number of values in that field. If the field contains a single value, this function returns 1 . If the field has no values, this function returns NULL. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title4) |
| `mvdedup` | `mvdedup(<mv>)` | This function takes a multivalue field and returns a multivalue field with its duplicate values removed. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title5) |
| `mvfilter` | `mvfilter(<predicate>)` | This function filters a multivalue field based on an arbitrary Boolean expression. The Boolean expression can reference ONLY ONE field at a time. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title6) |
| `mvfind` | `mvfind(<mv>,<regex>)` | This function tries to find a value in the multivalue field that matches the regular expression. If a match exists, the index of the first matching value is returned (beginning with zero). If no values match, NULL is returned. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title7) |

| `mvjoin` | `mvjoin(<mv>,<delim>)` | This function takes two arguments, a multivalue field and a string delimiter. The function concatenates the individual values within <mv> using the value of <delim> as a separator. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title9) |
| `mvmap` | `mvmap(<mv>,<expression>)` | This function iterates over the values of a multivalue field, performs an operation using the <expression> on each value, and returns a multivalue field with the list of results. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title10) |
| `mvrange` | `mvrange(<start>,<end>,<step>)` | This function creates a multivalue field for a range of numbers. This function can contain up to three arguments: a starting number, an ending number (which is excluded from the field), and an optional step increment. If the increment is a timespan such as 7d , the starting and ending numbers are treated as UNIX time. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title11) |
| `mvreverse` | `mvreverse(<value>)` | The mvreverse command reverses the order of the values in a multivalue field. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title12) |
| `mvsort` | `mvsort(<mv>)` | This function uses a multivalue field and returns a multivalue field with the values sorted lexicographically. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title13) |
| `mvzip` | `mvzip(<mv_left>,<mv_right>,<delim>)` | This function combines the values in two multivalue fields. | `mv_to_json_array` | `mv_to_json_array(<field>, <infer_types>)` | This function maps the elements of a multivalue field to a JSON array. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title15) |
| `split` | `split(<str>,<delim>)` | This function splits the string values on the delimiter and returns the string values as a multivalue field. | eval, fieldformat, where | [docs](https://help.splunk.com/en/splunk-enterprise/spl-search-reference/10.2/evaluation-functions/multivalue-eval-functions#ariaid-title16) |
